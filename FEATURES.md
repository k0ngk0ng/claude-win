# Features

## Core Features

- **Native Windows Client**: Built with WPF for a true Windows-native experience
- **Bundled Node.js Runtime**: No need to install Node.js separately
- **Auto-Install Claude Code CLI**: Automatically installs Claude Code on first run
- **Persistent Session**: Maintains conversation context across messages

## User Interface

- **Slash Command Toolbar**: Quick access to common commands (/help, /clear, /compact, /resume, /memory, /mcp, /config, /cost, /doctor, /init, /review)
- **Image Upload Support**: Add images via button or Ctrl+V paste
- **Command History**: Navigate previous commands with Up/Down arrows
- **Terminal-Style Output**: Clean, readable output display

## Configuration

- **Visual Settings Panel**: Configure all options through a friendly UI
- **JSON Import**: Import configuration from JSON format
- **Environment Variables**: Full support for ANTHROPIC_* environment variables
  - API Key / Auth Token
  - Base URL
  - Model selection (main model, small fast model)
  - Default models (Haiku, Sonnet, Opus)
  - Proxy settings
  - Timeout and retry configuration

## Security

- **Permission Skip Option**: Optional `--dangerously-skip-permissions` flag
- **Process Isolation**: Proper process management and cleanup

## Installation Options

- **Installer (Recommended)**: Full setup with bundled Node.js
- **Portable Version**: Self-contained, no installation required
- **Framework-Dependent**: Lightweight, requires .NET 8 Runtime

## System Requirements

- Windows 10 version 1809 (Build 17763) or later
- Windows 11 supported
- 64-bit system required
